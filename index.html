Що я вчив?

HTML

Використовується для розміщення елементів на сторінці. Створює "скелет" сайту, де кожен елемент має своє місце і призначення.

CSS

Використовується для оформлення веб-сторінки. З його допомогою можна змінювати кольори, шрифти, відступи, розміри та інші параметри елементів сторінки

JavaScript

Мова програмування, завдяки якій сторінки стають динамічними, дозволяючи користувачам взаємодіяти з її елементами.
